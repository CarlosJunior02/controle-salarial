# 💰 FinTrack - Dashboard de Controle Financeiro

O **FinTrack** é uma aplicação web desenvolvida para auxiliar usuários no gerenciamento de suas finanças pessoais. A plataforma permite registrar receitas e despesas, acompanhar o saldo atual, visualizar gráficos de gastos e monitorar metas de economia de forma simples e intuitiva.

---

## 🚀 Funcionalidades

- ✅ Cadastro de receitas e despesas
- 📊 Dashboard financeiro
- 💵 Cálculo automático do saldo
- 🎯 Meta mensal de economia
- 📈 Gráfico de gastos por categoria
- 🔍 Pesquisa de transações
- 🎨 Alternância entre tema claro e escuro
- 📱 Layout totalmente responsivo
- 🗑️ Exclusão de transações

---

## 📊 Funcionalidades do Dashboard

### Dashboard Principal

- Saldo Atual
- Receitas do mês
- Despesas do mês
- Meta de economia

### Cadastro de Transações

Cada transação possui:

- Descrição
- Valor
- Tipo (Receita ou Despesa)
- Categoria
- Data

### Histórico

- Lista completa das transações
- Busca por descrição
- Filtro por tipo
- Exclusão de registros

### Gráfico

Visualização dos gastos separados por categorias utilizando Chart.js.
